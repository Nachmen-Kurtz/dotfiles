* Summary

Reference for the complete software stack required by each window manager
environment in this dotfiles repo. Covers both Wayland (Sway, Hyprland) and
X11 (i3, Qtile) setups.

* Install Hints

Install all Wayland (Sway) deps on Void Linux:

#+BEGIN_SRC sh
  sudo xbps-install sway waybar fuzzel mako swaylock swayidle \
    wlsunset grim slurp wl-clipboard brightnessctl NetworkManager alacritty
#+END_SRC

Install all Wayland (Hyprland) deps on Void Linux:

#+BEGIN_SRC sh
  sudo xbps-install hyprland waybar fuzzel swaync hypridle hyprpaper \
    hyprsunset swayosd grim slurp wl-clipboard brightnessctl kitty
#+END_SRC

Install all X11 (i3) deps on Void Linux:

#+BEGIN_SRC sh
  sudo xbps-install xorg i3 i3status rofi dunst i3lock \
    xss-lock dex setxkbmap flameshot NetworkManager \
    pipewire alacritty
#+END_SRC

Install all X11 (Qtile) deps on Void Linux:

#+BEGIN_SRC sh
  sudo xbps-install xorg qtile dunst setxkbmap numlockx \
    flameshot NetworkManager pipewire alacritty
#+END_SRC
